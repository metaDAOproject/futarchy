# 缺点
但当然，未来学治理并不完美。这种方法可能存在一些潜在的缺陷。

### 凯恩斯美人竞赛
有时候，投资者会购买他们认为其他人会买的东西，而不是他们认为基本面支持的东西。 [GameStop逼空事件](https://en.wikipedia.org/wiki/GameStop\_short\_squeeze) 就是一个例子，在加密货币中可以找到更多这样的例子。

一个潜在的解决方案是使用白名单市场。投资者需要申请成为未来市场的一部分，证明他们过去是基于基本面进行交易的。

### 条件市场是零和的
Futarchy 使用条件市场，这些市场是零和的：你在市场上的任何收益都是其他人的损失。由于聪明的市场参与者不会参与零和游戏，这将限制流动性和对 futarchic 市场的参与。

您可以通过提供激励来促进市场交易或提供流动性来解决此问题。由于futarchy会产生正外部性（良好的决策），这种激励是可持续的。

### 可扩展性
Futarchy 适用于重大决策，例如是否解雇 CEO，但不太适用于较小的决策，例如是否解雇部门负责人，因为这些行动通常对股价的影响微不足道。

MetaDAO的白皮书中描述了这个问题的解决方案。
