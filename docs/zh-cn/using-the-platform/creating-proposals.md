# 创建提案
[MetaDAO简介](https://blog.metadao.fi/a-futards-guide-to-the-meta-dao-7a6b8d66443a)

[说明博客文章](https://blog.metadao.fi/so-you-want-to-raise-a-proposal-2d83304c0b9d)

商业项目是MetaDAO将金融资本转化为创收产品的方式。商业直接行动在这些产品上运作，通过调整参数来追求客户满意度和盈利能力。运营项目和直接行动支持业务，确保MetaDAO拥有合适的人才和资源来创造新产品和管理现有产品。

这四种提案类型中的每一种都有其自己的模板。它们列在这里：

{% content-ref url="../proposal-templates/business-direct-action.md" %} 商业直接行动 {% endcontent-ref %}
{% content-ref url="../proposal-templates/operations-project.md" %} 运营项目 {% endcontent-ref %}
{% content-ref url="../proposal-templates/operations-direct-action.md" %} 操作直接行动 {% endcontent-ref %}

项目提案通常应由对DAO负责其执行的企业家提出。任何人都可以提出直接行动提案。

您可以使用任何文档应用程序来创建提案。
